# 自动阅卷系统

基于 AI 的自动化阅卷工具，支持截图识别、AI 评分、自动填分全流程。

## 功能特点

- **截图识别**：自动截取题目区域，调用 AI 视觉模型识别和评分
- **智能评分**：支持多家 AI 服务商（OpenAI、智谱 AI、通义千问、豆包、百度千帆、科大讯飞、小米 MiMo 等），自定义评分标准
- **思考模式**：自动为支持思考模式的模型（MiMo、Qwen3、DeepSeek-R1、QwQ）启用深度推理，提高评分准确率
- **结构化输出**：AI 以「最终得分 + 反馈信息」分隔符格式输出，分数提取不受反馈内容干扰
- **多区域框选**：支持框选多个答案区域（适用于填空题），自动拼接为带编号标签的一张图，防止答案错位
- **区分印刷与手写**：提示词指导 AI 区分答题卡印刷文字和学生手写内容，只以手写部分为评分依据
- **自动生成评分标准**：支持从截图或上传题目图片，由 AI 自动生成评分标准
- **规则调优**：收集 AI 评分记录并关联截图，人工标记正确分数后利用大模型分析偏差模式，生成优化规则，一键应用
- **快捷优化建议**：输入优化想法，AI 分析后直接生成新评分标准
- **自动填分**：基于坐标点击的自动填分，支持分数输入框、提交按钮、下一题按钮
- **空白卷检测**：基于截图灰度波动识别空白答卷，支持直接数值滑块调节和手动标记空白卷自动校准阈值
- **批量处理**：支持批量阅卷，可设置总份数并自动停止
- **评分失败自动停止**：API 超时或异常时自动停止阅卷，不会误填 0 分

## 使用方法

### 启动程序

```bash
python 上层GUI.py
```

### 配置步骤

1. **选择 AI 服务商**：从下拉列表选择，自动填入 base_url 和建议模型
2. **填写 API Key**：标准服务商直接填写，百度千帆用 `API_Key:Secret_Key`，科大讯飞用 `appId:apiKey:apiSecret`，小米 MiMo 用 `tp-xxxxx`
3. **设置评分标准**：手动输入或通过「生成」菜单自动生成
4. **截图配置（菜单栏 → 截图配置）**
   - **选择截图区域**：单区域框选，适用于大多数题型
   - **多区域框选（填空题）**：分别框选每个答案空位，自动拼接为带编号标签的图片
     - 操作：拖拽框选 → Enter 确认 / Backspace 撤销 / Esc 取消
   - 选择分数输入框、提交按钮、下一题按钮位置
   - 主界面「准备状态」会显示各项是否已设置
5. **开始阅卷**：单题模式处理当前题目，批量模式按设定份数循环处理

### 空白卷检测

- **滑块调节**：主界面「空白阈值」滑块范围 0~40，值越高越容易判定为空白
- **测试空白检测**：截取当前区域，显示灰度波动和判定结果
- **标记空白卷**：翻到空白答题卡页面，点击「标记空白卷」，程序自动识别灰度波动并设置阈值（+2 余量）

### 规则调优

1. 每次评分后自动收集记录（含截图和 AI 反馈信息）
2. 鼠标悬停在记录行上可预览 AI 分数、AI 反馈信息和截图
3. 选中记录，填写正确分数并点击「标记」
4. 收集偏差记录后，点击「规则调优」调用大模型分析错误模式
5. 点击「应用新规则」将优化后的规则写入评分标准

- 小米 MiMo（mimo-v2.5、mimo-v2.5-pro）
- 通义千问 Qwen3 系列
- DeepSeek-R1 系列
- QwQ 系列

### 小米 MiMo 说明

- **API Key 格式**：`tp-xxxxx`（Token Plan 专用）
- **认证方式**：使用 `api-key` 请求头（非标准 Bearer token）
- **集群选择**：中国 / 新加坡 / 欧洲，base_url 分别为 `https://token-plan-{cn|sgp|ams}.xiaomimimo.com/v1`

## AI 输出格式

评分指令要求 AI 按以下结构化格式输出：

```
最终得分：4分
===反馈开始===
第1题得2分，第2题得1分，第3题得1分。
===反馈结束===
```

- 第一行为得分（唯一分数提取来源）
- 反馈信息用于调试，不会干扰分数提取
- 有分隔符时禁止从全文兜底匹配，防止误提

## 配置文件

配置文件 `config.json` 已从 Git 跟踪中排除，包含以下字段：

```json
{
  "provider": "OpenAI",
  "api_key": "your_api_key",
  "model": "gpt-4o-mini",
  "base_url": "https://api.openai.com",
  "criteria": "请根据标准答案评分",
  "batch_mode": false,
  "total_questions": "0",
  "blank_threshold": 15.0,
  "screenshot_region_norm": [0.1, 0.1, 0.9, 0.8],
  "score_input_pos": [100, 200],
  "submit_btn_pos": [180, 200],
  "next_btn_pos": [260, 200]
}
```

截图区域和按钮位置选择后不会自动保存，需要点击「文件 → 保存配置」后才会持久化。

## 故障排除

| 问题 | 解决方案 |
|------|----------|
| API 超时 | 检查网络连接，超时后阅卷自动停止，不会误填 0 分 |
| 评分提取错误 | AI 以结构化格式输出，第一行为得分，反馈信息不会干扰 |
| 空白卷误判 | 点击「标记空白卷」校准阈值，或手动调节滑块 |
| 填空题答案错位 | 使用「多区域框选」，每个空位单独框选，带编号标签 |
| 印刷文字被当作答案 | 提示词已指导 AI 区分印刷文字和手写内容 |
| 填分失败 | 检查按钮位置配置 |
